# Staff Kiosk (React + Vite)

This repository contains a small React + Vite scaffold for the Staff Kiosk app.

Quick start (assuming Node.js and npm are installed):

```powershell
cd "c:\Users\Khalijosh\Documents\staff-kiosk"
npm install
npm run dev
```

- Development server: http://localhost:5173
- Build: `npm run build`
- Preview build: `npm run preview`

Files:
- `src/` — React source files (entry: `src/main.jsx`, root component: `src/App.jsx`)
- `index.html` — app HTML
- `vite.config.js` — Vite config
- `package.json` — scripts and dependencies

If Node/npm are not installed, download them from https://nodejs.org/ and retry.
